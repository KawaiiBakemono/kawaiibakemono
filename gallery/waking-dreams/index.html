About the Project

Waking Dreams is an RPG Maker game lead by close friend, DreamingCassette. The game follows a troubled young girl, Aina, who's hometown has fallen under a mysterious illness, causing many to go into a strange coma. It's up to her to get to the bottom of it, for better or for worse.
I've been a part of the game's development from the very beginning, being chosen as one of the select few to play the very first alpha of the game all the way back at the beginning of 2024.

Plugins

As of writing this, I've created 2 custom JavaScript plugins for the game at the request of DreamingCassette. I made one for the game's title screen, and another that adds customizeable borders.

Dynamic Title Screen Plugin

The title screen was a bit bland, so given a concept of DreamingCassette's vision for a more fleshed-out one, I went to work creating a custom plugin for her to use in-engine.
The plugin allows for multiple layers of images, all of which can be adjusted, scaled, parallaxed, and even animated using image atlases. Not only that, but the title screen can be changed instantly in-game with the use of RPG Maker events to coencide with the game's story, which will then be saved to a JSON file that's loaded in when the game boots.

Dynamic Borders Plugin

Since most RPG Maker games (including this one) don't use a 16:9 aspect ratio, it leaves boring black borders on either side of the game window. To spice it up, I added in the ability to use custom images for borders to fill in that void. While simpler, these too can be dynamically changed in-game with the use of events.

Logo Design

The original logo

First concepts. I was given a few requirements for the design, like a symbol that represents the game (I chose the tulip).

Here I experimented with a smoother style using SVGs, but DreamingCassette wasn't a fan of the look, so I went back to pixel art. We also settled on using a lavender palette instead of the red.

After throwing around some more ideas, this is the final logo we settled on.
